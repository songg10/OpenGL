Basic Ray Tracer by Song Tung Nguyen (301354423)
*Instructions:
- The program runs on Linux OS and requires GLEW and freeglut installed.
- freeglut: Type "sudo apt-get install freeglut3-dev" on terminal.
- GLEW: Type "sudo apt-get install libglew-dev" on terminal or follow the instructions on "http://glew.sourceforge.net/index.html"
- To run the program, please head into the folder "Assignment 3" and type "make" (ignores all the warnings) and type "./rayT [scene#]" to run the program.
- Scene 1: Demonstartes basic phong lightning controls. Type "./rayT 1" after "make".
- Scene 2: Demonstrates that reflection and shadowing are working. Type "./rayT 2" after "make".
- Scene 3: Demonstrates that refraction are working (reflection and shadowing are disabled). Type "./rayT 3" after "make".
- Scene 4: Customized scene which included all the features with some additional features such as super sampling.

*Requirements:
- All of the requirements are fulfilled.

*Known bugs:
- There are currently bleed out effect in the picture of the first and third scene. If you run them directly from the program, the bleed out effect will be eliminated.
